# 图书管理系统 python398

## 项目概述

图书管理系统是一款基于Python Flask框架开发的Web应用，旨在提供简洁、易用的图书和用户数据管理。

## 技术栈

### 后端
- **Python Flask**：轻量级Web框架，用于处理路由、请求和响应。

### 前端
- **HTML5/CSS3**：页面结构和样式。
- **Bootstrap 5**：响应式布局和UI组件。
- **JavaScript**：交互逻辑和表单验证。

### 数据存储
- **CSV文件**：使用逗号分隔值文件存储图书和用户数据，无需数据库服务器。

## 系统角色
- 系统管理员：管理图书和用户信息。
- 普通用户：查询图书信息，进行图书借阅。

## 开发环境
- **Python 3.9**
- **Flask库**

## 功能模块
- 用户注册与登录
- 图书信息展示
- 图书借阅
- 用户信息管理
- 图书信息管理

## 运行环境
- 操作系统：兼容Windows、Linux和macOS。
- 环境依赖：Python 3.9及以上版本，Flask库。
